Getting Started

Our API provides access to the complete Mr Games collection with simple HTTP requests. You'll need an API key to get started.

1. Get Your API Key

Sign up for a free account to get your API key:

POST https://api.mrgames.pro/v1/auth/register
Content-Type: application/json

{
  "email": "your@email.com",
  "password": "yourpassword"
}

2. Make Your First Request

Once you have your API key, you can start making requests:

GET https://api.mrgames.pro/v1/games
Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

API Endpoints

GET /v1/games

Retrieve a list of all available games with pagination support.

Parameters

  • page - Page number (default: 1)
  • limit - Items per page (default: 20)
  • category - Filter by category
  • search - Search term

Example Request

GET https://api.mrgames.pro/v1/games?page=1&limit=5&category=action
Authorization: Bearer YOUR_API_KEY
